# 轻松让您的机器人实现变现

想让您的智能机器人为您创造收益？Callgent 提供简单、灵活的变现方案！只需几分钟配置，您的 API 服务即可开始盈利！

## 🚀 3 个步骤快速开启变现

### 1. 选择 Callgent
（从 Callgent 列表中选择一个服务作为起点。）

### 2. 设置认证
配置认证以确保服务安全，并指定扣款方式：开启锁表示由创建者扣费，设置价格表示由调用者付费，未来还将支持指定扣款用户。

![认证设置](/img/tutorials/auth.jpg)

### 3. 启用变现
在图一**test**下找到您的 API，点击锁状图标，选择认证方式，确认锁已关闭即表示设置成功。

## 💸 灵活的计费模式

根据您的服务类型选择最适合的计费方式，操作简单且透明。

![计费选项](/img/tutorials/pricing.jpg)

### 方式一：按次计费（适合新手）
为每次 API 调用设定固定费用，适合标准化的服务。

- **适用场景**：固定、 predictable 的查询服务
- **示例**：每次调用收费 $0.01
- **设置方法**：在计费字段输入金额即可

### 方式二：按 Token 计费（适合内容服务）
根据 Token 使用量计费，适合文本生成、翻译等动态内容服务。

- **适用场景**：输出长度可变的服务
- **示例代码**：
```javascript
function calcPrice(req) {
  // 每个 Token 收费 $0.0002
  return req.tokenUsage * 0.0002;
}
```

## 🔗 测试您的服务
设置完成后，您即可调用已变现的服务。了解详情：[调用 Callgent](/docs/quick-start/call-the-callgent)
