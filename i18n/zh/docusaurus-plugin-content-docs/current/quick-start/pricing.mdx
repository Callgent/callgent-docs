### 2. 设置认证
配置认证以确保服务安全，并指定扣款方式，不同类型的锁表示不同作用。

Client Entries：用于身份认证，如果锁是打开状态，表示创建者付费，关闭则是调用者付费。

Service Adaptors：用于表示本次请求如何进行付费。

## 💸 我们有灵活的计费模式

![计费选项](/img/tutorials/pricing.jpg)

### 方式一：按次计费（适合新手）
为每次 API 调用设定固定费用，适合标准化的服务。

- **适用场景**：固定、 predictable 的查询服务
- **示例**：每次调用收费 $0.01
- **设置方法**：在计费字段输入金额即可

### 方式二：按 Token 计费（适合内容服务）
根据 Token 使用量计费，适合文本生成、翻译等动态内容服务。

- **适用场景**：输出长度可变的服务
- **示例代码**：
```javascript
// 每个 Token 收费 $0.0002
function calcPrice(req) {
  return req.tokenUsage * 0.0002;
}
```
